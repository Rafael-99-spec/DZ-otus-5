Описание ДЗ

Vagrantfile, находящийся в данном репозитории предназначен для развертывания виртуальных машин nfsserver и nfsclient. 

После развертывания стенда на nfsserver разворачивается nfs-server с настроенной сетевой папкой - /shared/upload, при помощи секции SHELL, в которой выполняются bash-скрипты. 

Аналогичным образом на nfsclient создаются два systemd unit-a network-online.service и shared-upload.mount. Они предназначены для составления дерева зависимостей NetworkManager->network-online.service->shared-upload.mount. 

Главной целью данного дерева зависимостей является автоматическое монтирование сетевой папки /mount/upload (который находится на nfsserver) на /shared/upload, непосредственно после поднятия сетевых интерфейсов на клиентской ОС.

Два файла конфигурации предназначенные для systemd юнитов также расположены в репозитории в виде текстовых файлов.
